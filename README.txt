
README - Sitio Web del Parque Nacional Podocarpus
--------------------------------------------------

Este archivo contiene los elementos necesarios para visualizar una página web informativa
sobre el Parque Nacional Podocarpus. Fue desarrollada por estudiantes para fines educativos.

Contenido del ZIP:
- index.html        → Página principal del sitio.
- styles.css        → Estilos visuales del sitio web.
- /images/          → Carpeta con imágenes de ejemplo usadas en la web.

INSTRUCCIONES DE USO:
1. Extrae el archivo ZIP en tu computadora.
2. Abre el archivo "index.html" en cualquier navegador web moderno.
3. Asegúrate de que la carpeta "images" esté en la misma ubicación que el HTML para que las imágenes se muestren correctamente.
